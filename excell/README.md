# ☕ Coffee Orders Dashboard

## 📌 Project Overview

The **Coffee Orders Dashboard** is a data visualization project designed to analyze coffee sales and customer ordering patterns. The dashboard provides insights into **total sales, top customers, product performance, and country-wise orders** using interactive visualizations.

The main objective of this project is to transform raw order data into **clear, visual insights** that help understand business performance and customer behavior.

---

# 📂 Dataset Information

The dataset includes multiple tables used to build the dashboard:

* **Orders** – Contains order details such as product, quantity, and sales.
* **Customers** – Customer information including name and location.
* **Products** – Details about coffee products and pricing.
* **Total Sales** – Aggregated sales data used for visualization.
* **Country** – Country-wise order distribution.
* **Top 5 Customers** – Customers with the highest purchase value.

These tables are connected to provide meaningful insights through the dashboard.

---

# 📊 Dashboard Features

The dashboard includes the following key visualizations:

### 📈 Total Sales Analysis

Displays the overall revenue generated from coffee orders over time.

### 🌍 Country-wise Sales

Shows which countries generate the most coffee sales.

### 👥 Top 5 Customers

Identifies the customers who contribute the most to total revenue.

### ☕ Product Performance

Analyzes which coffee products are ordered the most.

### 📅 Order Trends

Shows how coffee sales change across different time periods.

# 🔎 Data Analysis Process

## 1️⃣ Data Collection

The raw data was collected from multiple tables including **orders, customers, and products**.

## 2️⃣ Data Cleaning

Data was cleaned by:

* Removing duplicate values
* Handling missing data
* Formatting numeric and date fields

## 3️⃣ Data Transformation

Tables were connected using **common keys such as customer ID and product ID** to create relationships.

## 4️⃣ Dashboard Creation

Interactive charts and visual elements were used to represent:

* Sales trends
* Customer contributions
* Product popularity
* Geographic distribution of orders
